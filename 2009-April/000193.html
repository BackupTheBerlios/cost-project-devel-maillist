You are not using a full path so COSTcore is likely to be on a "Rlib" 
folder under the folder where you run the install command.

Try something like

R CMD INSTALL -l ~/Rlib COSTcore_1.2-05.tar.gz

and load the library with

library(COSTcore,lib.loc="~/Rlib")

or else use a full path to where you want to install.

A more important problem. We've decided to develop for the latest 
version of R, if your library will be written in R261 it will not run on 
the latest versions (there were some important changes since) which 
means that to use your library users will have to have installed on 
their computers two R versions and keep in 261 the old versions of COST 
packages. It will be a mess !!

Besides you will not be able to use the latest enhancements of COSTcore 
and you maybe using an old data structure.
