Hi David,

I use COSTcore and COSTeda without problems in linux ... you must have 
the latest R version and maybe you'll need to download the source code 
from CVS and compile it yourself.

Regarding compiling in windows the best is to use a windows machine and 
compile it there. There are some cross compilation manuals 
(cran.r-project.org/doc/contrib/cross-build.pdf). You should take a look 
at the R manual (http://cran.r-project.org/doc/manuals/R-exts.pdf).
