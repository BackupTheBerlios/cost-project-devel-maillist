You're hitting version problems which are very difficult to sort out 
now. The version you're using is for R251 and you're getting a problem 
with namespaces, which have changed in R.

The latest stable version of COSTcore "one" is 126 which runs on R271.

The newest version is COSTcore133 that runs on R281. All packages should 
be developed for this platform !! Otherwise we may get into serious 
trouble regarding maintenance and for users it will be hell, if they'll 
need to use several R versions with several COST package versions to run 
each analysis.
