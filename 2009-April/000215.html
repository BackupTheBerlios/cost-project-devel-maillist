I've a method to produce a FLQuant from a dbeOutput object. It works fine.

Load FLCore, copy paste the code attached, run the examples on the manual
(we need the lanEstim to test) and run

flq <- FLQuant(lanEstim)

You can check the results

head(lanEstim at ageStruc)
